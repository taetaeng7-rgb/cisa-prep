# cisa-prep
https://taetaeng7-rgb.github.io/cisa-prep/
개인 학습용 CISA 수험 퀴즈 웹앱 (모바일 우선, 정적 웹앱).

- 본 저장소에는 앱 코드만 포함됩니다. 학습 콘텐츠(문항 데이터)는 암호화된 번들(`data/questions.enc`)로만 존재하며, 소유자 본인만 복호하여 사용합니다.
- 본 앱은 ISACA와 무관한 개인 학습 도구이며, 열람 가능한 형태의 교재 원문이나 실제 시험 문항을 포함하지 않습니다.
- 개인 사용 목적의 프로젝트로, 콘텐츠에 대한 이용 문의는 받지 않습니다.
